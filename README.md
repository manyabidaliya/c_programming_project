# **MUSIC GENRE SUGGESTER**

## FEATURES
- USER FRIENDLY
- EASY TO USE
- SUGGEST SONGS


## Technologies Used
- C
  
## USAGE
1. ENTER YOUR PREFERRED GENRE OF MUSIC.
2. TO CONTINUE ENTER 1 AND REPEAT STEP 1.
3. TOP SONGS OF SELECTED GENRE WILL BE DISPLAYED.
4. TO EXIT THE PROGRAM ENTER 0.
